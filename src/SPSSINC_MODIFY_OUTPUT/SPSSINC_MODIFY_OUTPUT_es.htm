Modificar títulos de resultados

Utilice este cuadro de diálogo para modificar los títulos y/o el texto en el panel de titulares del Visor y los títulos o el texto de los propios elementos. También puede modificar su visibilidad. Puede especificar los tipos de elementos que se procesarán y los criterios de selección en el cuadro de diálogo principal y las acciones que se aplicarán en el subcuadro de diálogo Acciones. Las operaciones se aplican a la ventana del visor designada.

Seleccione uno o más tipos de elementos que se procesarán. La mayoría son evidentes. Los encabezados son los elementos de titulares que tienen hijos por debajo. Los títulos son los títulos de los elementos reales. No todos los elementos tienen títulos que se puedan modificar con este comando. En concreto, los títulos de los elementos de gráficos, modelos y árboles no se pueden modificar. Sin embargo, su texto de titular sí se puede modificar.

De forma predeterminada, el comando procesa el resultado del procedimiento inmediatamente anterior (ignorando los elementos de registro que intervienen). Puede cambiarlo para procesar el contenido actual del visor.

Sólo se procesan los elementos visibles. Se pueden ocultar los elementos seleccionados eligiendo Ocultar en el subcuadro de diálogo Acciones.

Criterios de selección

Se pueden limitar los elementos procesados especificando una o varias de las opciones siguientes: Comando, Subtipo de tabla, Criterios de titulares y Criterios de título del elemento. Los campos que no se aplican a un tipo de objeto particular se ignoran al determinar la selección. Sólo se modifican los elementos que cumplan todos los criterios aplicables.

Los campos Comandos y Subtipo de tabla se corresponden con los identificadores SGR de los elementos. Puede encontrarlos pulsando con el botón derecho del ratón en la entrada del titular de un ejemplo o desde Utilidades > Identificadores SGR.

Criterios de titulares y Criterios de texto de elemento se pueden especificar como coincidencias exactas (ignorando la distinción entre mayúsculas y minúsculas) o por el texto de inicio o de fin. También puede utilizar una expresión regular, que es una forma general de describir un patrón. Las expresiones regulares no se explican aquí. Un recurso muy útil es http://www.amk.ca/python/howto/regex.

Subcuadro de diálogo Acciones

Pulse en el botón Acciones para especificar qué hacer con los elementos seleccionados. Si no especifica nada en Acciones, el comando no hará nada.

Sustitución de texto

Puede introducir texto nuevo para el titular y/o título de elemento o texto en los grupos Texto de titular y Texto de elemento. Puede ser texto literal para sustituir el contenido anterior o una expresión regular de sustitución.

En el texto nuevo, puede hacer referencia al texto anterior, además de incluir una letra o número de secuencia. La letra o el número de secuencia se representa mediante "\\0" y el texto anterior mediante "\\1". Por ejemplo, para añadir un número de secuencia al título anterior, puede introducir
Tabla \\0: \\1
También puede utilizar la notación \\0 en una expresión regular.

Puede utilizar la notación de formato html o rtf en el texto de sustitución. Por ejemplo,
<html>El <i>nuevo</i> título
haría que "nuevo" apareciese en cursiva.

En los criterios de selección, la información de formato se ignora. La directiva \\1 también omite la información de formato.

Si utilizó paréntesis en la expresión regular de la selección, puede usar \n referencias a estos grupos, donde n es el número de grupo. Si no se ha utilizado ninguna expresión regular para la selección, el título anterior se considera grupo 1.

Numeración de secuencia

El valor de secuencia puede ser un número, una letra o un número romano. Marque la casilla Números o letras para seleccionarlo. De manera predeterminada, los estilos son numéricos y empiezan por el 1. Puede introducir un valor de inicio diferente. Si tiene una o más letras (hasta cuatro), las letras de la secuencia coincidirán en mayúsculas y minúsculas con la primera letra. Si selecciona uno de los botones de números romanos, la secuencia estará en números romanos. El valor de inicio, si se introduce, debe ser numérico en este caso.

Para obtener valores de secuencia, debe marcar la casilla Números o letras e incluir \\0 en el titular o el texto del elemento.

Introducción de saltos de página

Si selecciona Títulos en el grupo Objetos de resultado del cuadro de diálogo principal, puede introducir un salto de página delante de cada título que cumpla los criterios de selección. Marque la casilla Saltos de página. De forma predeterminada, esto sólo afecta a los títulos del nivel de titulares indentado un nivel por debajo de un encabezado de procedimiento. Establezca el botón Todo para establecer saltos en objetos de título de todos los niveles de titulares.

Funciones personalizadas

Los programadores de Python y sus amigos pueden especificar la llamada de una función personalizada para cada objeto seleccionado. Esto puede ahorrar el tener que escribir el código para llegar a los elementos adecuados. Introduzca la función que se aplicará como
nombre módulo.nombre función

Las funciones personalizadas pueden tener parámetros. Para especificar uno o más parámetros, escriba los nombres y valores de parámetros e incluya la lista de parámetros entre paréntesis. Por ejemplo,
misfunciones.escribirtituloselementos(file='filespec')
llamaría a la función escribirtituloselementos del módulo misfunciones. Consulte la documentación de cualquier función personalizada en particular para conocer qué parámetros acepta. Los nombres de parámetros deben escribirse siguiendo las mayúsculas y minúsculas especificadas en la definición de función.

Si el valor del parámetro es una cadena, inclúyala entre comillas simples, por ejemplo
option='ambos'

Características adicionales

Este cuadro de diálogo genera la sintaxis del comando de extensión SPSSINC MODIFY TABLES. Para mostrar la ayuda de este comando, ejecute la siguiente sintaxis:

SPSSINC MODIFY TABLES /HELP.

La sintaxis del comando le permite:
  • Especificar una lista de comandos para su selección
  • Especificar una lista de subtipos de tabla
  • Procesar los elementos de resultado visibles, no visibles o todos ellos
  • Eliminar elementos seleccionados en lugar de sólo ocultarlos

Requisitos

Este cuadro de diálogo requiere el complemento de integración de Python y el comando de extensión SPSSINC MODIFY TABLES. Para IBM SPSS Statistics 19 y versiones posteriores, el complemento y el comando de extensión se instalan con el paquete Essentials for Python. Si desea más información, consulte Cómo obtener complementos de integración, en Sistema principal > Preguntas más frecuentes en el sistema de ayuda de IBM SPSS Statistics. Si ha descargado SPSSINC MODIFY TABLES de la Comunidad de SPSS, siga las instrucciones del archivo léame asociado.
