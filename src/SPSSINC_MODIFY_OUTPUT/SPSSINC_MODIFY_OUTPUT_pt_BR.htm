Modificar títulos de saída

Use essa caixa de diálogo para modificar os títulos e/ou texto na área de janela da estrutura de tópicos do Visualizador e os títulos ou texto dos próprios itens. Também é possível alterar a visibilidade desses itens. Especifique os tipos de itens a serem processados e os critérios de seleção na caixa de diálogo principal. Use o subdiálogo Ações para especificar as ações a serem aplicadas. As operações são aplicadas na janela do Visualizador designada.

Escolha um ou mais tipos de itens a serem processados. A maior parte deles é evidente. Cabeçalhos são os itens de estrutura de tópicos que têm itens filhos. Títulos são os títulos dos itens reais. Nem todos os itens têm títulos que podem ser modificados por esse comando. Especificamente, títulos de itens como Gráficos, Modelos e Árvores não podem ser alterados. No entanto, é possível modificar o texto de estrutura de tópicos.

Por padrão, o comando processa a saída do procedimento imediatamente anterior (ignorando os itens de log intervenientes). É possível alterar essa configuração para processar todo o conteúdo atual do Visualizador.

Somente os itens visíveis são processados. É possível ocultar os itens selecionados escolhendo a opção Ocultar no subdiálogo Ações.

Critérios de seleção

É possível limitar os itens processados especificando uma ou mais opções do Comando, Subtipo de tabela, Critérios de título de estrutura de tópicos e Critérios de título de item. Os campos que não se aplicam a um tipo específico de objeto são ignorados na determinação da seleção. Somente os itens que satisfazem a todos os critérios aplicáveis são modificados.

Os campos Comando e Subtipo de tabela correspondem aos identificadores de OMS para os itens. Para localizá-los, clique com o botão direito na entrada de estrutura de tópicos de um exemplo ou clique em Utilitários>Identificadores de OMS.

As opções Critérios de título de estrutura de tópicos e Critérios de texto de item podem ser especificadas como correspondências exatas (não diferenciando maiúsculas de minúsculas) ou pelo texto inicial ou final. Também é possível usar uma expressão regular, que é uma maneira geral de descrever um padrão. As expressões regulares não são explicadas aqui. Uma origem útil é http://www.amk.ca/python/howto/regex.

Subdiálogo Ações

Clique no botão Ações para especificar o que fazer aos itens selecionados. Se nada for especificado em Ações, o comando não fará nada silenciosamente.

Substituindo texto

É possível inserir um novo texto para a estrutura de tópicos e/ou para o título ou texto de item nos grupos Texto de estrutura de tópicos e Texto de item. O texto pode ser um texto literal para substituir o conteúdo antigo, ou uma expressão regular de substituição.

No novo texto, é possível fazer referência ao texto antigo e incluir um número ou letra de sequência. O número ou letra de sequência é representado por "\\0" e o texto antigo por "\\1". Por exemplo, para incluir um número de sequência no título antigo, é possível inserir
Tabela \\0: \\1
É possível também usar a notação \\0 em uma expressão regular.

É possível usar a notação de formatação html ou rtf no texto de substituição. Por exemplo:
<html>The <i>New</i> Title
faria com que "New" aparecesse em itático.

Nos critérios de seleção, as informações de formatação são ignoradas. A diretiva \\1 também omite as informações de formatação.

Se você usou parênteses na expressão regular de seleção, poderá usar referências \n a esses grupos, em que n é o número do grupo. Se nenhuma expressão regular tiver sido usada para a seleção, o título antigo será considerado grupo 1.

Número de sequência

O valor de sequência pode ser um numero, letras ou numerais romanos. Marque a caixa Itens de número ou de letra para escolher. Por padrão, os estilos são numéricos e iniciam com 1. É possível inserir um valor inicial diferente. Se o valor for uma ou mais letras (até quatro), a sequência será formada por letras que seguem o padrão de maiúsculas/minúsculas da primeira letra. Se você marcar um dos botões de numeral romano, a sequência será em números romanos. Neste caso, o valor inicial, se inserido, deve ser numérico.

Para obter os valores de sequência, você deve marcar a caixa Itens de número ou de letra e incluir \\0 na estrutura de tópicos ou no texto do item.

Inserindo quebras de página

Se você selecionar Títulos no grupo Objetos de saída na caixa de diálogo principal, será possível inserir uma quebra de página na frente de cada título que corresponda aos critérios de seleção. Marque a caixa Quebras de página. Por padrão, são afetados somente títulos no nível de estrutura de tópicos recuados um nível abaixo de um cabeçalho de procedimento. Defina o botão Todos para ajustar as quebras em objetos de título em todos os níveis de estrutura de tópicos.

Funções customizadas

Programadores Python e seus amigos podem especificar uma função personalizada a ser chamada para cada objeto selecionado. Isso pode evitar a escrita de código para navegar até os itens apropriados. Insira a função a ser aplicada como
nome name.function do módulo

Funções customizadas podem ter parâmetros. Para especificar um ou mais parâmetros, coloque a lista de parâmetros entre parênteses e escreva os nomes e valores do parâmetro. Por exemplo:
myfunctions.writeitemtitles(file='filespec')
chamaria a função writeitemtitles no módulo myfunctions. Consulte a documentação sobre qualquer função customizada específica para saber quais parâmetros ela aceita. Os nomes de parâmetro devem ser escritos seguindo o padrão de maiúsculas e minúsculas especificado na definição de função.

Se o valor de parâmetro for uma sequência, coloque-o entre aspas simples, por exemplo,
option='both'

Recursos adicionais

Essa caixa de diálogo gera sintaxe para o comando de extensão SPSSINC MODIFY TABLES. Para exibir ajuda para este comando, execute a seguinte sintaxe:

SPSSINC MODIFY TABLES /HELP.

A sintaxe de comando permite:
  • Especificar uma lista de comandos para seleção
  • Especificar uma lista de subtipos de tabela
  • Processar itens de saída visíveis, não visíveis ou todos
  • Excluir os itens selecionados em vez de apenas ocultá-los

Requisitos

Essa caixa de diálogo requer o plug-in de integração para Python e o comando de extensão SPSSINC MODIFY TABLES. Para o IBM SPSS Statistics 19 e posterior, o Plug-in e o comando de extensão são instalados com o pacote Essentials for Python. Para obter informações adicionais, consulte Como Obter Integration Plug-In, no Sistema Principal > Perguntas mais Frequentes no sistema de ajuda do IBM SPSS Statistics Help. Se você transferiu por download SPSSINC MODIFY TABLES da Comunidade SPSS, siga as instruções no arquivo leia-me associado.
