修改输出标题

使用此对话框可以修改查看器大纲窗格中的标题和/或文本以及项本身的标题或文本。您还可以更改其可视性。您可以指定要处理的项类型以及主对话框中的选择条件以及要在操作子对话框中应用的操作。 操作将应用于指定的查看器窗口。

选择一个或多个要处理的项类型。大部分都是不言自明的。 标题是其下方有子代的大纲项。 题目是实际项的题目。并非所有项都有可由此命令修改的题目。尤其是，无法更改图表、模型和树项题目。但是，可以更改其大纲文本。

缺省情况下，此命令处理来自紧邻的先前过程的输出（忽略中间的日志项）。您可以将其更改为处理查看器的所有当前内容。

将仅处理可视项。您可以通过在“操作”子对话框中选择隐藏来隐藏选定项。

选择条件

您可以通过指定“命令”、“表子类型”、“大纲标题条件”和“项标题条件”中的一个或多个来限制处理的项。 在确定选择时将忽略不适用于特定对象类型的字段。将仅修改满足所有适用条件的项。

命令和表子类型字段对应于项的 OMS 标识。您可以通过右键单击示例的大纲条目或从实用程序 > OMS 标识来查找这些内容。

大纲标题条件和项文本条件可指定为完全匹配（忽略大小写），或者按起始或结束文本匹配。您还可以使用正则表达式，这是描述模式的一般方式。 正则表达式不在此处进行解释。一个有用的资料来源是 http://www.amk.ca/python/howto/regex。

操作子对话框

单击操作按钮以指定要对选定项执行的操作。 如果在“操作”中未指定任何内容，该命令将保持静默，不执行任何操作。

替换文本

您可以在大纲文本和项文本组中为大纲和/或项标题或文本输入新文本。这可以是用于替换旧内容的字面值文本，也可以是替换正则表达式。

在新文本中，可以引用旧文本，并且可以包含序号或字母。序号或字母由 "\\0" 表示，旧文本由 "\\1" 表示。例如，要将序号添加到旧标题，您可以输入
表 \\0: \\1
您还可以在正则表达式中使用 \\0 表示法。

您可以在替换文本中使用 html 或 rtf 格式化表示法。例如，
<html><i>新</i>标题
会导致“新”以斜体样式显示。

在选择条件中，将忽略格式化信息。\\1 伪指令也会省略格式化信息。

如果在选择正则表达式中使用了括号，那么可以使用对这些组的 \n 引用，其中 n 是组号。如果没有使用正则表达式进行选择，那么旧标题被视为组 1。

序列编号

序列值可以是数字、字母或罗马数字。选中数字或字母项框以进行选择。缺省情况下，样式为数字，从 1 开始。您可以输入不同的起始值。如果是一个或多个字母（最多 4 个字母），那么序列将是与第一个字母的大小写匹配的字母。 如果您选中其中一个罗马数字按钮，该序列将以罗马数字表示。在此情况下，如果输入起始值，该值必须是数字。

为了获取序列值，您必须同时选中数字或字母项框并在大纲或项文本中包含 \\0。

插入分页符

如果在主对话框上的输出对象组中选择标题，那么可以在符合选择条件的每个标题前面插入分页符。选中分页符框。缺省情况下，仅大纲级别在过程标题下缩进一个级别的题目会受到影响。设置全部按钮以在所有大纲级别的标题对象上设置中断。

定制函数

Python 程序员及其朋友可以指定要为每个选定对象调用的定制函数。这样就不用编写代码来浏览到相应的项。将要应用的函数输入为
模块名称.函数名称

定制函数可以有参数。要指定一个或多个参数，请将参数列表括在圆括号内，并编写参数名称和值。 例如，
myfunctions.writeitemtitles(file='filespec')
将调用 myfunctions 模块中的函数 writeitemtitles。 请查阅任何特定定制函数的文档以了解它接受的参数。参数名称必须按函数定义中所指定的大小写进行书写。

如果参数值是字符串，请将其括在单引号内，例如：
option='both'

其他功能部件

此对话框将生成 SPSSINC MODIFY TABLES 扩展命令的语法。要显示此命令的帮助，请运行以下语法：

SPSSINC MODIFY TABLES /HELP。

命令语法允许您：
  • 指定用于选择的命令的列表
  • 指定表子类型的列表
  • 处理可视、不可视或所有输出项
  • 删除选定项，而不是仅仅将其隐藏

要求

此对话框需要 Integration Plug-In for Python 和 SPSSINC MODIFY TABLES 扩展命令。对于 IBM SPSS Statistics 19 及更高版本而言，此 Plug-In 和扩展命令随 Essentials for Python 程序包一起安装。有关更多信息，请参阅“IBM SPSS Statistics 帮助”系统的“核心系统”>“常见问题”中的“如何获取集成插件”。如果您是从 SPSS Community 下载 SPSSINC MODIFY TABLES，请按相关自述文件中的指示信息执行操作。
