出力タイトルの変更

このダイアログを使用して、ビューア アウトライン ペインのタイトルまたはテキスト (あるいはその両方)、項目自体のタイトルまたはテキスト (あるいはその両方) を変更します。表示するかどうかを変更することもできます。処理する項目の種類および選択基準をメイン ダイアログで指定し、適用する操作を「操作」サブダイアログで指定します。操作は、指定したビューア ウィンドウに適用されます。

処理する 1 つまたは複数の項目タイプを選択します。ほとんどは、見たとおりの項目です。見出しはアウトライン項目であり、下部に子があります。タイトルは、実際の項目のタイトルです。すべての項目のタイトルがこのコマンドで変更できるわけではありません。特に図表、モデル、ツリーの項目タイトルは変更できません。ただし、アウトライン テキストの項目は変更できます。

デフォルトでは、このコマンドは直前の手順で作成された出力を処理します (途中のログ項目は無視します)。コマンドを変更して、ビューアの現在のすべての内容を処理できます。

処理されるのは表示可能な項目だけです。「操作」サブダイアログで「非表示」を選択して、選択した項目を非表示にすることができます。

選択基準

コマンド、テーブルのサブタイプ、アウトライン タイトルの基準、および項目タイトルの基準のうち 1 つまたは複数を指定して、処理する項目を制限できます。特定のオブジェクト タイプに適用されないフィールドは、選択の決定時には無視されます。該当するすべての基準を満たす項目だけが変更されます。

「コマンド」フィールドおよび「テーブルのサブタイプ」フィールドは、項目の OMS 識別子に該当します。例のアウトライン エントリを右クリック、または「ユーティリティ」→「OMS 識別子」を選択して、これらの項目を表示できます。

「アウトライン タイトルの基準」および「項目テキストの基準」を完全一致 (大文字と小文字の区別は無視) としてまたは開始テキストおよび終了テキストによって指定できます。パターンを説明する一般的な方法である正規表現も使用できます。ここでは、正規表現については説明しません。役立つ情報として http://www.amk.ca/python/howto/regex を参照してください。

「操作」サブダイアログ

選択した項目に対して何を行うかを指定するには「操作」ボタンをクリックします。「操作」で何も指定しない場合、コマンドは何も実行しません。

置換テキスト

「アウトライン テキスト」および「項目テキスト」グループのアウトラインまたは項目 (あるいは両方) のタイトルまたはテキストに、新しいテキストを入力できます。以前の内容を置き換えるリテラル テキストにすることも、置換用の正規表現にすることもできます。

新しいテキストでは、以前のテキストを参照し、連続番号または文字を含めることができます。連続番号または文字は "\\0"、以前のテキストは "\\1" を使用して示します。例えば、古いタイトルに連続番号を追加するには、
Table \\0: \\1
と入力します。正規表現でも \\0 表記を使用できます。

置換テキストでは html 形式または rtf 形式の表記を使用できます。たとえば、
<html><i>新しい</i> タイトル
と入力すると、「新しい」が斜体で表示されます。

選択基準では、書式の情報は無視されます。\\1 ディレクティブも書式情報を省略します。

選択の正規表現で括弧を使用した場合、これらのグループへの \n 参照を使用できます。n はグループ番号を示します。正規表現が選択に使用されなかった場合、古いタイトルはグループ 1 と見なされます。

連続番号

連続値には、数値、文字、ローマ数字を指定できます。選択するには、「数値または文字の項目」ボックスをオンにします。デフォルトでは、スタイルは数値で 1 から始まります。異なる開始値を入力できます。1 つまたは複数の文字 (最大 4 文字) である場合、連続番号の大文字または小文字は最初の文字と一致します。ローマ数字のボタンをオンにすると、連続番号がローマ数字になります。開始値を入力する場合、開始値は数値でなければなりません。

連続番号の値を取得するには、「数値または文字の項目」ボックスをオンにし、アウトライン テキストまたは項目テキストに \\0 を含めます。

ページ区切りの挿入

メイン ダイアログの「出力オブジェクト」グループの「タイトル」を選択すると、選択基準に適合する各タイトルの前にページ区切りを挿入できます。「ページ区切り」ボックスをオンにします。デフォルトでは、プロシージャの見出しの下で 1 レベル インデントしたアウトライン レベルのタイトルだけが影響を受けます。すべてのアウトライン レベルのタイトル オブジェクトに区切りを設定するには、「すべて」ボタンを設定します。

カスタム関数

Python プログラマは、選択したオブジェクトごとに呼び出すカスタム関数を指定できます。該当する項目に移動するコードを作成して保存できます。適用する関数を以下のように入力します。
module name.function name

カスタム関数にはパラメータを指定できます。1 つまたは複数のパラメータを指定するには、パラメータのリストを括弧で囲み、パラメータ名と値を入力します。たとえば、
myfunctions.writeitemtitles(file='filespec')
は、myfunction モジュールにある関数 writeitemtitles を呼び出します。 指定できるパラメータについては、特定のカスタム関数の資料を参照してください。パラメータ名は、関数定義で指定されたとおりに大文字と小文字を区別して記述する必要があります。

パラメータ値が文字列の場合、文字列を単一引用符で囲みます。例を示します。
option='both'

その他の機能

このダイアログは、SPSSINC MODIFY TABLES 拡張コマンドのシンタックスを生成します。このコマンドのヘルプを表示するには、次のシンタックスを実行します。

SPSSINC MODIFY TABLES /HELP

コマンド シンタックスを使用すると次のことが可能です。
  • 選択するコマンドのリストを指定する
  • テーブルのサブタイプのリストを指定する
  • 表示項目、非表示項目、またはすべての出力項目を処理する
  • 非表示にするのではなく、選択した項目を削除する

要件

このダイアログには Integration Plug-In for Python と SPSSINC MODIFY TABLES 拡張コマンドが必要です。IBM SPSS Statistics 19 以上の場合、Plug-in と拡張コマンドが Essentials for Python パッケージとともにインストールされます。詳しくは、IBM SPSS Statistics ヘルプ システムの「コア システム」>「よくある質問」の「Integration Plug-In の取得方法」を参照してください。 SPSS コミュニティーから SPSSINC MODIFY TABLES をダウンロードした場合、関連する readme ファイルの指示に従ってください。
